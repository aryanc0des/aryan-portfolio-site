Personal Portfolio Website;

A personal portfolio built from scratch using HTML5 and CSS3, designed to reflect my identity as a developer. It features a clean, terminal-inspired dark theme, a fully responsive layout using flexbox, and minimalistic UI elements with soft hover transitions. The project emphasizes learning by building — combining aesthetic design with semantic structure. Planned enhancements include typing animations and smooth fade-ins to create an engaging, developer-centric experience. This portfolio isn’t just a showcase — it’s a growing canvas for experimentation and meaningful creation;

Smart RFID Attendance System;

An IoT-based RFID Attendance System developed using Arduino, RFID modules, and Python integration for real-time data handling. The system logs student attendance by scanning RFID tags and instantly updates a Google Sheet using Google Apps Script and a Python bridge. Future enhancements include a student-parent dashboard app, professor admin portal, and automated alerts for absenteeism and performance tracking;
